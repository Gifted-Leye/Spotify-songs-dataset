**Excel Sample Dataset for Practicing Excel Functions**

This repository contains a sample dataset created solely for practice and learning purposes.
It is designed to help users understand and apply Excel functions and formulas effectively.

**Purpose**

The dataset is intended to:

Practice basic to advanced Excel functions

Improve data cleaning and transformation skills

Understand real-world data scenarios in Excel

Build confidence using formulas for analysis

**What You Can Practice With This Dataset**

Text functions (TRIM, PROPER, SUBSTITUTE, etc.)

Date and time functions

Lookup functions (VLOOKUP, XLOOKUP, INDEX/MATCH)

Logical functions (IF, AND, OR)

Data cleaning and validation

Basic data analysis and summaries

**Disclaimer**

This dataset is not real data and should be used strictly for learning, testing, and demonstration purposes.
